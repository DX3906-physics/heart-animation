# ❤️ 爱心动画

一个美丽的HTML5爱心动画，基于原始C++程序转换而来。

## 🌟 特性

- **心跳动画效果**：模拟真实的心跳节奏
- **多种颜色主题**：5种不同的颜色方案
- **交互控制**：
  - 🔄 重新开始动画
  - ⏯️ 暂停/继续播放
  - ⚡ 改变动画速度
  - 🌈 切换颜色主题
- **响应式设计**：适配各种屏幕尺寸
- **黑色背景**：高对比度的视觉效果

## 💝 动画原理

基于爱心参数方程：
```
x = 16 * sin³(t)
y = 13 * cos(t) - 5 * cos(2t) - 2 * cos(3t) - cos(4t)
```

## 🎯 如何使用

1. **在线访问**：直接点击上面的GitHub Pages链接
2. **本地打开**：下载后用浏览器打开 `index.html`
3. **交互操作**：
   - 按任意键重新开始动画
   - 点击按钮控制播放、速度、颜色
